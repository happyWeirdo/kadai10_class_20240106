# my_favorite_app
# 課題{10} -{Flutterで作るお気に入りのもの管理アプリ ログイン機能付き}-

## ①課題内容（どんな作品か）
- お気に入りのエンタメコンテンツやスポット、キャラクタのイメージを登録、削除、管理できるアプリです。
- メニュータブ、メインのPlaygroundページ、ユーザーのお気に入り一覧My Favoriteページを作りました。
- 一つずつのコンテンツ情報をタイルにまとめてあり、写真と簡単な説明分を作ってあります。
- 右下の追加ボタンを押すと、お気に入り登録でき、 My Favoriteページの一覧に追加されます。
- 一覧ページには削除機能を実装しています。

## ②工夫した点・こだわった点
- Flutterを使い、自分の作りたいアプリを意識して、作っています。
- UIのデザインはシンプルで使いやすい感じのものにしました。
- 色合いやUIの動きなど、ユーザーが見て楽しい、操作しやすいものを意識しました。


## ③難しかった点・次回トライしたいこと（又は機能）
- ログイン機能も実装したが、セキュリティとして、大丈夫かどうかをリサーチして、強化が必要だったら、改良したいです。
- DBとの接続方法がまだ知識としてないので、調べて今後は実装していきたいです。
